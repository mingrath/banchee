    Server Component that loads current business profile (if partially filled) and renders the wizard.

    ```typescript
    import { getCurrentUser } from "@/lib/auth"
    import { getBusinessProfile } from "@/models/business-profile"
    import { getSettings } from "@/models/settings"
    import SetupWizard from "./components/setup-wizard"

    export const dynamic = "force-dynamic"

    export default async function SetupPage() {
      const user = await getCurrentUser()
      const profile = await getBusinessProfile(user.id)
      const settings = await getSettings(user.id)
      return <SetupWizard initialProfile={profile} settings={settings} />
    }
    ```

    **1. Download THSarabunNew font files:**
    THSarabunNew is an open-source Thai font (free for commercial use). Download the Regular and Bold static TTF files and place them at:
    - `public/fonts/THSarabunNew.ttf` (Regular weight)
    - `public/fonts/THSarabunNew-Bold.ttf` (Bold weight)

    Source: Search GitHub for "THSarabunNew TTF" or use the f0nt.com download. The font MUST be static TTF format (not variable font) because @react-pdf/renderer does not support variable fonts.

    If direct download is not possible via CLI, create placeholder files with a TODO comment and a checkpoint note. However, prefer using curl/wget to download from a known GitHub source such as:
    `https://github.com/ArtifexSoftware/fontoforget/raw/main/THSarabunNew/THSarabunNew.ttf` (or similar open-source repo).

    Alternative: Search npm for "thsarabunnew" font package.

    **2. Font registration (exports/pdf/fonts.ts):**
    Create the font registration module per RESEARCH.md:

    ```typescript
    import { Font } from "@react-pdf/renderer"
    import path from "path"

    export function registerThaiFonts() {
      Font.register({
        family: "THSarabunNew",
        fonts: [
          {
            src: path.join(process.cwd(), "public/fonts/THSarabunNew.ttf"),
            fontWeight: "normal",
          },
          {
            src: path.join(process.cwd(), "public/fonts/THSarabunNew-Bold.ttf"),
            fontWeight: "bold",
          },
        ],
      })
    }

    // Call registration at module load
    registerThaiFonts()
    ```
