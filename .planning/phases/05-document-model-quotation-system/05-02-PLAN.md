    Create app/(app)/apps/quotation/actions.ts with three server actions.

    The file must start with `"use server"` directive.

    **createQuotationAction(prevState: ActionState&lt;QuotationData&gt; | null, formData: FormData): Promise&lt;ActionState&lt;QuotationData&gt;&gt;**

    Steps (clone pattern from tax-invoice/actions.ts):
    1. Get current user via `getCurrentUser()` from `@/lib/auth`
    2. Parse scalar fields from formData using `Object.fromEntries(formData.entries())`
    3. Validate with `quotationFormSchema.safeParse()` -- if invalid, return `{ success: false, error: parsed.error.errors[0].message }`
    4. Parse line items from FormData dynamic arrays:
       ```typescript
       const itemDescriptions = formData.getAll("item_description") as string[]
       const itemQuantities = formData.getAll("item_quantity") as string[]
       const itemUnits = formData.getAll("item_unit") as string[]
       const itemUnitPrices = formData.getAll("item_unitPrice") as string[]
       const itemDiscounts = formData.getAll("item_discount") as string[]
       ```
    5. Validate at least 1 line item with description -- if empty, return error "\u0e01\u0e23\u0e38\u0e13\u0e32\u0e40\u0e1e\u0e34\u0e48\u0e21\u0e2d\u0e22\u0e48\u0e32\u0e07\u0e19\u0e49\u0e2d\u0e22 1 \u0e23\u0e32\u0e22\u0e01\u0e32\u0e23"
    6. Map line items to QuotationLineItem[] with baht-to-satang conversion:
       ```typescript
       const items: QuotationLineItem[] = itemDescriptions.map((desc, i) => {
         const qty = parseFloat(itemQuantities[i] || "0")
         const unitPrice = Math.round(parseFloat(itemUnitPrices[i] || "0") * 100) // baht to satang
         const discount = Math.round(parseFloat(itemDiscounts[i] || "0") * 100)   // baht to satang
         const amount = Math.round(qty * unitPrice) - discount
         return {
           description: desc,
           quantity: qty,
           unit: itemUnits[i] || "\u0e0a\u0e34\u0e49\u0e19",
           unitPrice,
           discount,
           amount,
         }
       })
       ```
    7. Compute totals:
       ```typescript
       const subtotal = items.reduce((sum, item) => sum + item.amount, 0)
       const overallDiscount = Math.round(parsed.data.overallDiscount * 100) // baht to satang
       const subtotalAfterDiscount = subtotal - overallDiscount
       const includeVat = parsed.data.includeVat === "true"
       const vatResult = includeVat ? computeVATOnSubtotal(subtotalAfterDiscount) : { vatAmount: 0, total: subtotalAfterDiscount, subtotal: subtotalAfterDiscount }
       const vatRate = includeVat ? VAT_RATE : 0
       const total = subtotalAfterDiscount + vatResult.vatAmount
       ```
    8. Load business profile via `getBusinessProfile(user.id)` for seller snapshot
    9. Load contact via `getContactById(user.id, parsed.data.contactId)` for buyer snapshot -- if null, return error "\u0e44\u0e21\u0e48\u0e1e\u0e1a\u0e1c\u0e39\u0e49\u0e15\u0e34\u0e14\u0e15\u0e48\u0e2d"
    10. Compute validUntil: `addDays(new Date(parsed.data.issuedAt), parsed.data.validityDays)` using date-fns
    11. Create document via `createDocument(user.id, { ... })` with documentType "QUOTATION"
    12. Build QuotationData response object from the created document
    13. Call `revalidatePath("/apps/quotation")` to refresh the page
    14. Return `{ success: true, data: quotationData }`
    15. Wrap in try/catch -- on error, log and return `{ success: false, error: "\u0e2a\u0e23\u0e49\u0e32\u0e07\u0e43\u0e1a\u0e40\u0e2a\u0e19\u0e2d\u0e23\u0e32\u0e04\u0e32\u0e44\u0e21\u0e48\u0e2a\u0e33\u0e40\u0e23\u0e47\u0e08" }`

    3. Call `updateDocumentStatus(user.id, documentId, newStatus)`
